#AI Test-cases Generator

AI Test Cases Generator is a web-based application developed as a diploma project. The system automatically generates structured test cases and WebdriverIO automation code based on a provided website URL. It analyzes the page structure and user interface elements, then uses artificial intelligence to produce relevant QA documentation and test scripts.
The application is designed to simplify and speed up the testing process for QA engineers by reducing manual effort. With a clean and intuitive interface, users can easily generate, copy, and export test cases or automation code, making the tool suitable for both manual and automation testing workflows.
